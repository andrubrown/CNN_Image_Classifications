%%%%%%%%%% Installation %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
1, Install CUDA from developer.nvidia.com/cuda-downloads

2, Install ATLSA: sudo apt-get install libatlas-base-dev

3, pip install -r /path/to/caffe/python/requirements.txt
